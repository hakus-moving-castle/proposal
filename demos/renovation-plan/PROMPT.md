# Renovation Plan — 重生版 UI/UX

## Prompt（版面規格 — 2025 重生版）

Premium Dark 預設（#0B0F19）+ Polar Light 切換。Tailwind CDN + vanilla JS + CSS Variables。

### UI/UX 鐵律

1. **全域高質感色系**：Premium Dark / Polar Light，霓虹綠/深海藍/Linear 紫點綴
2. **極致呼吸感**：段落 ≤3 行，卡片 padding 2.5rem，區塊 spacing 24
3. **全面表格化**：痛點、盤點、Token、Skills 一律 HTML table
4. **策略性折疊**：僅 TS/JSON/BDD/目錄樹等技術規格進 `<details>`
5. **核心大方露出**：Mermaid、四角色 I/O、Token 表、i18n 鐵律
6. **重點標示**：凍結點、i18n 鐵律、Pro plan 警告用 callout-emerald/amber

### 8 Section 架構

1. s1 — 前情提要痛點表格 + Vision 對照
2. s2 — 頂層架構（Mermaid 露出 + 目錄折疊 table）
3. s3 — 四角色 Skills Workflow（I/O 露出 + 規格折疊）
4. s4 — 核心考古盤點策略 table
5. s5 — 變更管理 & Main Docs 凍結
6. s6 — Figma 三層 Token（table 露出 + 組織/對齊折疊）
7. s7 — i18n 全自動管線（鐵律露出 + JSON/步驟折疊）
8. s8 — 驗收 + Skills table + Issue 範例 + Roadmap

三語支援 (zh-tw / en / ja)。無 build step，雙擊 index.html 可開啟。
